## Features

- **Next.js** (App Router) with React 19 and TypeScript for pages, components, and API routes
- **MongoDB** for persistent data

## Getting Started

1. Clone the repository:

    ```sh
    git clone https://github.com/KimNattanan/cee-final.git
    cd cee-final
    ```

2. Install dependencies:

    ```sh
    npm install
    ```

3. Configure environment variables

    Copy `.env.example` to `.env` and set the required variables (see `.env.example` for keys).

4. Run the development server:

    ```sh
    npm run dev
    ```
